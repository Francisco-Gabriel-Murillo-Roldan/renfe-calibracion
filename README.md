## 📌 Resumen

**Español:** Calibración paramétrica de 11 locomotoras diésel RENFE para Open Rails mediante **ethical reverse engineering**, documentación técnica real y pruebas empíricas estandarizadas (Método Teluroncio).

**English:** Parametric calibration of 11 RENFE diesel locomotives for Open Rails through **ethical reverse engineering**, real technical documentation, and standardised empirical testing (Teluroncio Method).
